# Schneider Electric MCBラインナップ（2026年2月時点）

## 概要
Schneider Electricは「9 Series」ブランドの下にMCB製品群を展開している。
住宅向け（Resi9）、産業・商業向け（Acti9）、OEM向け（Multi9）、
新興国向け（Easy9/Domae）と、ターゲット市場ごとに明確にシリーズを分けている。

## Acti9 iC60シリーズ（産業・商業向けフラッグシップ）

### iC60N（スタンダード）
- 規格: IEC 60947-2 / IEC 60898-1 デュアル
- Icu @220-240V: 36kA / @380-415V: 20kA / @440V: 10kA（IEC 60947-2）
- Icn @230V: 6000A / @400V: 6000A（IEC 60898-1）
- In: 0.5A〜63A
- カーブ: B, C, D
- 極数: 1P, 1P+N, 2P, 3P, 3P+N, 4P
- Uimp: 6kV / Ui: 500V
- 1極幅: 9mm（DINレール取付幅 18mm/極）
- 端子: ねじ式ケージ端子（EverLink対応モデルあり）
- 最大接続電線: 25mm²
- 使用温度: -25°C〜+70°C
- スマート機能: VisiTrip, VisiSafe, PowerTag対応
- アクセサリー: Vigi iC60（漏電ブロック）, 補助接点, シャントトリップ, モーターオペレータ, iARC（AFDD）

### iC60H（高遮断容量）
- Icu @220-240V: 30kA / @380-415V: 30kA / @440V: 15kA
- Icn @230V: 10000A / @400V: 10000A
- その他仕様はiC60Nと同等
- ターゲット: 短絡電流レベルが高い産業設備・変圧器近傍

### iC60L（超高遮断容量）
- Icu @220-240V: 100kA / @380-415V: 100kA / @440V: 25kA（IEC 60947-2）
- Icn @230V: 15000A（IEC 60898-1）
- In: 0.5A〜63A
- カーブ: B, C, D, K, Z（KとZカーブはiC60Lのみ対応）
- ターゲット: 大規模産業プラント・変電設備直近

### iC40N（コスト最適化）
- Icu @380-415V: 10kA / Icn: 6000A
- In: 1A〜63A / カーブ: B, C
- ターゲット: コスト重視の商業・軽工業

### C60H-DC（DC専用）
- DC専用設計
- 最大220VDC（2極直列時）/ 125VDC（1極時）
- Icu: 10kA @220VDC
- 太陽光発電・蓄電池・EV充電設備向け

## Multi9 C60SPシリーズ（OEM・グローバル）
- UL489 / CSA C22.2 No.5 / IEC 60947-2 マルチ規格対応
- Icu @480Y/277V: 10kA（UL489基準）
- Icu @415V: 15kA（IEC 60947-2基準）
- In: 0.5A〜63A / カーブ: B, C, D
- 極数: 1P, 2P, 3P, 4P
- 「same polarity」「opposite polarity」両対応（UL489）
- ターゲット: 北米輸出向け制御盤、装置メーカー（OEM）

## Resi9シリーズ（先進国住宅）
- IEC 60898-1 / BS EN 60898-1 専用
- Icn: 6000A〜10000A（サブモデルによる）
- In: 6A〜40A / カーブ: B, C
- 極数: 1P, 1P+N, 2P
- プラグオンバスバー対応で施工時間を大幅短縮
- ターゲット: 欧州・先進国の一般住宅・集合住宅

## Easy9シリーズ（新興国向けベーシック）
- IEC 60898-1 専用
- Icn: 3000A〜6000A（地域バリエーションあり。エジプト向け3000A、ベトナム向け6000A等）
- In: 6A〜63A / カーブ: B, C
- 極数: 1P, 1P+N, 2P, 3P, 4P
- 最小限の機能でコスト最適化
- ターゲット: 東南アジア・中東・アフリカ・南アジアの住宅・小規模商業

## Domaeシリーズ（リージョナルエコノミー）
- IEC 60898-1 専用
- Icn: 6000A
- In: 6A〜40A / カーブ: C のみ
- 極数: 1P, 1P+N, 2P, 3P, 4P
- 特定地域（主に東南アジア）限定の超コスト重視モデル
- Cカーブのみに絞ることで部品点数とコストを削減
