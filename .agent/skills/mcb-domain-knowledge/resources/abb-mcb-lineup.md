# ABB MCBラインナップ（2026年2月時点）

## 概要
ABBは「System pro M compact」ブランドの下にMCB製品群を展開。
産業用（S200シリーズ）、住宅用（SH200シリーズ）、高性能用（S800シリーズ）、
北米向け（SU200Mシリーズ）とターゲットを分けている。

## S200シリーズ（産業・商業向けスタンダード）
- 規格: IEC 60947-2 / IEC 60898-1 デュアル
- Icu @400V: 25kA（S200）/ 15kA（S200M）（IEC 60947-2）
- Icn: 6000A〜10000A（IEC 60898-1）
- In: 0.5A〜63A
- カーブ: B, C, D, K, Z
- 極数: 1P, 1P+N, 2P, 3P, 3P+N, 4P
- ポジション: Schneider Acti9 iC60Nの直接対抗

### S200Pシリーズ（高遮断容量）
- Icu @400V: 25kA
- Icn: 15000A〜25000A
- ポジション: Schneider Acti9 iC60H/Lの対抗

## S200 M UCシリーズ（AC/DC兼用）
- AC/DC兼用設計（Universal Current）
- IEC 60947-2 準拠
- Icu @60VDC: 10kA
- In: 0.5A〜63A / カーブ: B, C, D, K, Z
- ポジション: Schneider C60H-DCの対抗、かつAC/DC兼用で差別化

## S200 MTシリーズ（鉄道・トラクション向け）
- 鉄道車両・トラクション用途特化
- EN 50155（鉄道規格）準拠
- ニッチ市場向けの特殊モデル

## SH200シリーズ（住宅向け）
- IEC 60898-1 専用
- Icn: 6000A
- In: 6A〜63A / カーブ: B, C
- 極数: 1P, 1P+N, 2P, 3P, 4P
- ポジション: Schneider Resi9/Easy9の対抗

## SU200Mシリーズ（北米OEM向け）
- UL489 / CSA 認証
- Icu @480Y/277V: 10kA
- In: 0.5A〜63A / カーブ: B, C, D
- ポジション: Schneider Multi9 C60SPの直接対抗

## S800シリーズ（高性能MCB — ハイエンド）
- IEC 60947-2 専用（産業用の上位クラス）
- Icu @400V: 最大50kA
- In: 10A〜125A（MCBとしては高定格）
- カーブ: B, C, D, K, UC（DC対応）
- ポジション: Schneider Acti9 iC60Lの対抗。定格電流125Aまでカバーする点で差別化
- 重工業・再生可能エネルギー・船舶向け
