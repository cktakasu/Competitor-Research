# MCB ドメイン知識

## 規格の基本

### IEC 60898-1（住宅・小規模商業向け）
- 一般使用者（ordinary persons）による操作を想定
- 遮断容量は Icn（A単位）で表記
- 定格電流は 125A 以下
- 代表的な遮断容量レンジ: 1500A, 3000A, 4500A, 6000A, 10000A, 15000A, 20000A, 25000A

### IEC 60947-2（産業・商業向け）
- 電気技術者（instructed / skilled persons）による操作を想定
- 遮断容量は Icu（kA単位）で表記。Ics（使用短絡遮断容量）も定義される
- 定格電流の上限なし
- カテゴリA（非選択性）とカテゴリB（選択性）の区分あり

### ダブルスタンダード
- 同一製品が IEC 60898-1 と IEC 60947-2 の両方に適合する場合がある
- 例: Schneider Acti9 iC60N は IEC 60898-1 (Icn 6000A) と IEC 60947-2 (Icu 20kA@415V) の両方に適合
- これにより、住宅から産業まで幅広い用途に対応可能

## トリップ特性カーブ
- **B**: 3〜5×In で瞬時引外し。抵抗負荷（照明・ヒーター）向け
- **C**: 5〜10×In。汎用。小型モーター・一般回路向け
- **D**: 10〜20×In。大きな突入電流（モーター起動・変圧器）向け
- **K**: 8〜14×In。モーター保護特化
- **Z**: 2〜3×In。半導体・計装回路保護向け

## 主要スペック項目の定義
- **Icu**: 定格限界短絡遮断容量。この電流値まで遮断可能（ただし遮断後の再使用は保証しない）
- **Ics**: 定格使用短絡遮断容量。この電流値まで遮断後も継続使用可能
- **Icn**: IEC 60898-1における定格短絡遮断容量
- **Uimp**: 定格インパルス耐電圧（雷サージ等への耐性）
- **Ui**: 定格絶縁電圧
